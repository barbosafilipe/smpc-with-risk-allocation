# Exponential cone approach to joint chance constraints in stochastic model predictive control

Marques Barbosa, Filipe, and Johan Löfberg. *Exponential Cone Approach to Joint Chance Constraints in Stochastic Model Predictive Control.* International Journal of Control, 2025. doi:10.1080/00207179.2025.2492305.
